Proyecto final de labo_2

![LOGO UNSAM MUNDIAL](./imagenes/logo-UNSAM-mundial.jpg)

### Instalación:

  1) Versión Python 3.14.5
    * Descargable desde : https://www.python.org/downloads/release/python-3146/
    
    * Una vez instalado, ejecutar una actualizacion del pip con : python -m pip install --upgrade pip

  2) Instalar librerías:
      * pygame-ce
        - Correr en la consola : pip install pygame-ce
      * pygame_gui
        - Correr en la consola : pip install pygame-gui

  3) Descargar .Zip y extraer desde : https://github.com/SantiRautenberg/Isaac_TP_Final/archive/refs/heads/main.zip

  4) Ejecutar archivo "main.py"

### Cómo jugar:
  * Mover al jugador: W-A-S-D
  * Disparar: Flechas
  * Tecla ESC : Volver al menu inicial
